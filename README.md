# E-commerce kickstart

## Project structure

```shell
.
├── backend/ # API services
├── frontend/ # WEB services
├── services/ # docker services
├── docker-compose.yml
└── README.md
```

*****

## Start backend application

```shell
docker-compose up --build
```

*****

## Start frontend application

```shell
npm install
npm run serve
# npm run build
```
